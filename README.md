# 📊 ФинАнализ

Система анализа банковских выписок и учёта движения денежных средств (ДДС).

## Возможности

- Загрузка банковских выписок (Альфа-Банк, Сбербанк, форматы 1C/kl_to_1c, Excel, CSV)
- Автоматическая категоризация по справочнику контрагентов
- Отчёт ДДС с drill-down: статья → контрагент → документы
- Групповое редактирование категорий
- Синхронизация с Google Sheets через Apps Script
- Работает полностью в браузере, без сервера

## Быстрый старт

Открой приложение: **https://YOUR_USERNAME.github.io/finanaliz**

## Синхронизация с Google Sheets

1. Создай Apps Script на [script.google.com](https://script.google.com)
2. Вставь код из файла `ФинАнализ_AppsScript.js`
3. Опубликуй как веб-приложение (доступ: Все)
4. Вставь URL в Настройки → Интеграция с Google Sheets
